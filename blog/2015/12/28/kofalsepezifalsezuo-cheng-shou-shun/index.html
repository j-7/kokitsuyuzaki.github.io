このページの作成手順

SSHの設定

ssh-keygen -t rsa -C “My e-mail address of GitHub account” ssh-add ~/.ssh/id_rsa chmod 0600 ~/.ssh/id_rsa

GitHubのアカウントを作成（kokitsuyuzaki）

GitHub側で公開鍵を置く

pbcopy < ~/.ssh/id_rsa.pub # GitHubのSetting/SSH keys/にはりつけておく

GitHub上で、kokitsuyuzaki.github.ioというリポジトリを作成

octopressを取得

git clone git://github.com/imathis/octopress.git octopress cd octopress

インストール

bundle –path vendor/bundle bundle exec rake install

プレビュー

bundle exec rake preview # 試しに"http://localhost:4000/“にアクセスしてみる

初期設定

bundle exec rake setup_github_pages # “https://github.com/kokitsuyuzaki/kokitsuyuzaki.github.io"をコピペ

テーマをmnmlに変更する

git clone git://github.com/ioveracker/mnml.git .themes/mnml

rake install[‘mnml’]

rake generate

rake deploy

_config.yml, source以下のディレクトリ、ファイルを編集

GitHubにコミット

rake gen_deploy

以降は新しくブログを作成するたびに以下を実行

bundle exec rake new_post
